Queen Page

  • This project was another HTML and CSS practice, with this one focusing on “floats” and columns. The “random facts” about the band Queen are from https://ohfact.com/interesting-facts-about-band-queen/ , and each column repeats the same facts (we did this in order to save time and focus on styling instead of finding the perfect content). Cupcake Shop

  • This page was created in order to practice styling, more specifically CSS “classes.” For this particular page, I created two classes for altering text — “.flashy” and .”best” — and a third for altering the image of the “best cupcake.”

Cookie Recipe

  • This was a quick page that we made in class to practice “cloning” content from git hub to our own files. This page doesn’t have any CSS, but it does include different HTML elements such as: "h1" and "h2" tags; a random image; an external link to allrecipes.com; italic and bold text, and two different lists. Most of the recipe itself was supplied in the lesson, so I can’t take any credit even though I’m sure the cookies are delicious.
